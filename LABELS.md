# Label Taxonomy

Use these labels to keep the public tracker readable.

## Types

| Label | Color | Description |
| --- | --- | --- |
| `type: bug` | `d73a4a` | Public bug report or product defect. |
| `type: feature` | `a2eeef` | Public feature request or product improvement. |
| `type: known issue` | `fbca04` | Known issue published for visibility. |

## Status

| Label | Color | Description |
| --- | --- | --- |
| `status: needs triage` | `ededed` | New item awaiting review. |
| `status: under review` | `c5def5` | Being evaluated or clarified. |
| `status: planned` | `0e8a16` | Accepted for future work. |
| `status: in progress` | `1d76db` | Actively being worked on. |
| `status: released` | `5319e7` | Shipped or resolved in a released version. |
| `status: closed` | `6a737d` | Closed, duplicate, not planned, or no longer applicable. |

## Operations

| Label | Color | Description |
| --- | --- | --- |
| `duplicate` | `cfd3d7` | Duplicate of another public item. |
| `needs customer info` | `fef2c0` | More public-safe detail is needed. |
| `private support needed` | `b60205` | Needs a private support ticket instead of public tracking. |
