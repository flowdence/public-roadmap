# Flowdence Public Roadmap

This repository tracks public feature requests, known issues, and customer-visible improvements for Flowdence apps.

Use this repository when an item is safe to discuss publicly. For private support, security questions, licensing help, tenant-specific troubleshooting, or anything containing customer data, use [Flowdence Support](https://flowdence.io/support).

## Products

- ApprovalFlow for Confluence
- MuleSight for Confluence
- GrafanaSight for Confluence
- BoomiSight for Confluence
- Flowdence website and documentation

## What Belongs Here

- Public feature requests
- Public bug reports with sanitized reproduction steps
- Known issues that are safe to disclose
- Customer-visible roadmap items
- Released improvements that should remain discoverable

## What Does Not Belong Here

- Secrets, API tokens, credentials, or private URLs
- Marketplace SENs, invoices, procurement details, or billing records
- Customer names, emails, screenshots, logs, or tenant-specific data
- Security vulnerabilities or privacy concerns
- Urgent support requests

Open a private support ticket instead: https://flowdence.io/support

## Status Labels

- `status: needs triage` - submitted and not reviewed yet
- `status: under review` - being evaluated
- `status: planned` - accepted for a future release
- `status: in progress` - actively being worked on
- `status: released` - shipped
- `status: closed` - not planned, duplicate, or no longer applicable
